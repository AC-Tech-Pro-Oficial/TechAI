# TechDesign Creation Loop (SOP)

1.  **Concept Phase**:
    *   Analyze requirements.
    *   Consult `TechSearch` for trend inspiration if needed.
    *   Draft `design_spec.md`.

2.  **Asset Generation**:
    *   Generate Icons/Logos/Mockups using `generate_image`.
    *   Save to `D:\Images\{Project}`.

3.  **Frontend Handoff**:
    *   Update `tailwind.config.js` or `index.css` with Design Tokens.
    *   Instruct `TechCode` on component structure.

4.  **Polish Loop**:
    *   Review implemented UI.
    *   If not "Premium", iterate.
